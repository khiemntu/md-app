# MEVN-boilerplate
A fullstack boilerplate with Mongo, ExpressJS, VueJS and NodeJS.


## Setup Development Machine
1. Clone the repo `git clone git@git.dounets.com:khiemnguyen/display-app.git`

3. Open client
```
cd client
yarn
yarn serve
```

4. Open server
```
cd server
yarn start
```

5. Open `http://localhost:8080` in browser
